# Sistema-Sentinel
# 🛡️ Sentinel - Salvas Dados Automação CPF  Sistema simples de automação e salvamento de dados usando Python. O projeto permite inserir informações de teste (como nome e CPF fictício) e armazená-las localmente no navegador usando LocalStorage.  ## ⚙️ Funcionalidades - Salvar dados no navegador - Visualizar dados armazenados
